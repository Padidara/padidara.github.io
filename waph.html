Overview of lab1

Completing Lab 1 of the WAPH course gave a basic understanding of web technology. It addressed how to use Wireshark to analyze HTTP traffic, how to explore HTTP protocol intricacies using telnet, and how to design web applications. Among them were CGI programs and a simple PHP web application that processed user input. The lab also covered HTTP GET and POST requests, which are necessary for dynamic websites to function. Participants learned critical skills in protocol analysis and web development, as well as a thorough understanding of essential principles in HTTP communication.

Overview of Hackathon 1

This hackathon seeks to raise awareness about XSS attacks, understand code vulnerabilities, and adopt secure coding methods that follow OWASP principles to protect against cross-site scripting attacks. There are two tasks in the lab. Task 1 is to attack the URL http://waph-hackathon.eastus.cloudapp.azure.com/xss/, which contains six levels of vulnerabilities. Task 2 focuses on preventing XSS attacks using secure coding standards such as input validation and output sanitization.
